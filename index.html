    Hello there!

    In this section you'll find some instructions on how to use this calculator

    This calculator uses JavaScript math, so, if you already know it, you can probably skip this.

    • 1) There are some built-in functions, like sin, cos, factorial, etc.
    • 2) To create your own function, just follow this model: f = function(x) {return (2 * x)} and then just call it, like f(1) = 2
    • 3) When you do some calculation it will go to the list down bellow that you can move around and change its name
    • 4) You can also change the name of each individual calculation to keep track of what you're doing
